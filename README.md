# Highlit

Highlit is a simple yet powerful browser extension that allows users to highlight text on any webpage and save those highlights locally. 

## Features
- **Highlight & Save:** Select any text on a webpage and a convenient "Save Highlight" button will appear. 
- **Local Storage:** Your highlights are securely stored locally within your browser using Chrome extension storage.
- **Manage Your Highlights:** Click the extension's icon in the toolbar to view a list of all your saved texts.
- **Copy to Clipboard:** Easily copy any saved highlight to your clipboard with a single click.
- **Delete & Clear:** Remove specific highlights or clear all of them at once to keep your list organized.

## Installation
1. Clone or download this repository to your local machine.
2. Open Chrome (or a compatible Chromium-based browser) and navigate to `chrome://extensions/`.
3. Turn on **Developer mode** in the top right corner.
4. Click on **Load unpacked** and select the directory containing these project files.
5. The Highlit extension should now be installed and ready to use!

## How to Use
1. Highlight any text on any webpage.
2. A small "Save Highlight" button will appear near your cursor. Click it to save the text.
3. To view or manage your saved text, click the Highlit extension icon in your toolbar.
